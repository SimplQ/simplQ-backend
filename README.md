# SimplQ Backend

[SimplQ](https://simplq.me) is a completely web based queue management solution that anyone can use to create instant queues. 

### Development Envirmonment Setup Instructions

The project is written in Java, and deployed on AWS. These steps are to be followed when you are running the project for the first time.

1. Install Java 11 and maven. 
2. Clone this project
3. Build the jar:

```
cd complete/
mvn package
```
4. Run the jar:
```
java -jar target/rest-service-0.0.1-SNAPSHOT.jar
```
We follow Google's [Java Style Guidelines](https://github.com/google/styleguide). For Intellij, you can install the [google-java-format](https://plugins.jetbrains.com/plugin/8527-google-java-format) plugin and Eclipse users can set up the java style XML
